# /add-task — タスクマスターに新タスクを追加する

このプロジェクトの `app.js` には手続きタスクのマスターデータが定義されている。
新しいタスクを追加するときは以下の手順で行う。

## 手順

1. `app.js` を読み、タスクマスター配列（`TASK_MASTER` または同等の変数名）を特定する
2. `.claude/rules/data.md` のタスク構造を参照し、必須フィールドを確認する
3. 以下を確認してからタスクを追加する：
   - `id` が既存のものと重複していないか
   - `deps` に指定したIDが実際に存在するか
   - `cat` が定義済みカテゴリのいずれかか（'役所' | '銀行・金融' | '保険' | '免許' | '住居' | '会社' | 'カード・通信'）
   - `who` が '本人' | '妻' | '二人' のいずれかか
4. 追加後、`answers` の条件分岐（どの回答のときに表示するか）を確認・設定する
